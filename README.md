# Responsive Login Page

A responsive login interface with animated transitions, account registration, forgot password flow, and SQLite-backed authentication.

## Run locally

1. Open a terminal in this folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm start
   ```
4. Open `http://localhost:3000` in your browser.

## Features

- Landing page with animated login transition
- Login form with frontend and backend validation
- Register modal for new users
- Forgot password flow with email validation
- Dashboard after successful login
- SQLite database storage for user accounts
- Fully responsive layout for mobile and desktop
